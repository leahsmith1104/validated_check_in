Version 6

Hypotheses: people with duplicate initials need to be able to identify themselves.

Version 5

Hypotheses: Providing the user with a selection of names is the least laborious experience for them to identify themselves if their initials match that of anothers.

Version details: 5.0

5.0

  • Switches name question back to first name then surname
  • Adds a cognitive speed bump into surname question
  • Adds a confirmation screen for claimants to check the details they inputted before submitting
  • adds a third name question for duplications, asking to pick their name from 12.
  • turns back and start into text links again to the top of the screen
  • removed early fuctionality and provided SDC prompt on success page instead
